# OpenBot [Discord Bot]
An open sourced Discord bot, based on Discord.NET and NetCore 2.2.

## Usage: 
*Command list is available [here](https://dochacknik.keybase.pub/index/openbot/commands)*

## API Details: 
* Discord .NET C#: Version 2.0-beta
* DotNet Core 2.2-preview
